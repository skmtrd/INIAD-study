HTML elements we have learned

  • Describe the structure of the document
  • Apply styles to the text
  • Itemization
  • Linking
  • Image

The following are elements used when describing contents in the body element.
Do not forget to write document type declaration, html element, and head element.

Describe the structure of the document

h1 ~ h6

Represents that it is a heading.

p

Represents that it is a paragraph.

Apply styles to the text

b

Displays in bold style.

i

Displays in italic style.

strong

Emphasizes. Browser display becomes bold.

em

Emphasizes. Browser display becomes italic.

Itemization

ul

Combine with the li element to display unordered list.

ol

Combine with the li element to display ordered list.

li

Displays each item of list.

Linking

a

Displays the content of elements as links to another web page.

Image

img

Displays an image.
