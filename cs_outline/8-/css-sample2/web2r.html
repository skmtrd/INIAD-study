以下は、body要素内でコンテンツを記述する際に用いる要素である。
文書型宣言、html要素、head要素の記述を忘れないこと。

文章の構造を記述する

h1 ~ h6

見出しであることを表す。フレージングコンテンツ（独立したブロックを構成しない要素）内で用いないこと。

  • h1が最も上位で、h6が最も下位
  • 一般的にh1 ~ h6の順に小さくなる

p

段落であることを表す。フレージングコンテンツ内で用いないこと。

テキストを修飾する

b

太文字で表す。

i

イタリック体で表す。

strong

強調する。ブラウザの表示上は太文字になる。

em

強調する。ブラウザの表示上はイタリック体になる。

箇条書き

ul

li要素と組み合わせて順序のない箇条書きを表示する。フレージングコンテンツ内で用いないこと。子要素となるのは、li。

ol

li要素と組み合わせて順序のある箇条書きを表示する。フレージングコンテンツ内で用いないこと。子要素となるのは、li。

li

箇条書きの各項目を表示する。

リンク

a

要素を他のWebページへのリンクとして表示する。

  • href属性で、リンク先のページのURLを指定する。

画像

img

画像を表示する。空要素であることに注意。

  • src属性で表示対象の画像のURLを指定する。

表

table

tr, th, td要素と組み合わせて表を表示する。フレージングコンテンツ内で用いないこと。子要素となるのは、trなど。

tr

table要素の中で一行を表示する。子要素となるのは、thおよびtd。

th

tr要素の中で、ヘッダとなるセルを表示する。

td

tr要素の中で、データとなるセルを表示する。

フォーム

form

Webサーバに情報を送信するフォームを表示する。フレージングコンテンツ内で用いないこと。

button

ボタンを表示する。

input

フォーム用のコントロールを表示する。

textarea

複数行のテキストエリアを表示する。

select

選択式のメニューを表示する。子要素となるのは、option。

option

select要素の中で、個々の選択肢を表示する。

label

フォームのコントロールのラベルを表示する。

その他

hr

段落間の区切りを表示する。空要素であることに注意。フレージングコンテンツ内で用いないこと。

br

段落内で改行を表示する。空要素であることに注意。
